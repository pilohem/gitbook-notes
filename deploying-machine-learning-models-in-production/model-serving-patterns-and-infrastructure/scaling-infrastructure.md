# Scaling Infrastructure

Vertical scaling

* Increased Power
* More RAM
* Faster storage
* Adding or upgrading GPU/TPU

Horizontal scaling (scale up to your need - elasticity)

* More CPUs/GPUs instead of bigger ones
* Scale-up and down
